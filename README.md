# Cowin-Notifier

This is a python script which notifies you about available vaccination slots. It uses the API provided by CoWIN and fetches available slots for vaccination. 
You can input multiple pincodes and the number of days and it will notify you about the slots.

Pincode, number of days and search time can be changed from the script.

## Setup Guide
  
  1. Install requests (Used for make HTTP requests) 
     ```
     pip install requests
     ```
  2. Install Plyer (Used for sending desktop notifications)
     ```
     pip install plyer 
     ```
